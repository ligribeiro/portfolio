## Sobre

Este portfólio reúne os projetos acadêmicos e interdisciplinares (ABP) desenvolvidos ao longo do curso, com destaque para minha participação individual em cada entrega em grupo.

## Estrutura

```
.
├── index.html              # página principal
├── assets/
│   ├── css/style.css       # estilos (glassmorphism)
│   └── js/main.js          # interações (menu, contato)
└── README.md
```

## Como editar

1. Edite `index.html` para alterar textos, projetos ou seções.
2. Edite `assets/css/style.css` para alterar cores, fontes ou espaçamentos.
3. Edite `assets/js/main.js` para configurar o link do LinkedIn e o e-mail de contato (variáveis `LINKEDIN_URL` e `EMAIL` no topo do arquivo).

## Tecnologias

- HTML5
- CSS3 (glassmorphism, gradientes, responsivo)
- JavaScript (vanilla)
